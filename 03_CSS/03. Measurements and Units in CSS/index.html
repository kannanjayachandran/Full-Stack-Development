Css Units

  1. px - pixel absolute length
  2. em - Relative to font size
  3. rem - Relative to root element font size
  4. vw - 1% of viewport width
  5. vh - 1% of viewport height
  6. ch - width of "0" character
  7. % - usually Relative to parent value

Where to use

width and height

  1. % - to be Relative to parent
  2. vw/vh - to be Relative to viewport
  3. ch- for paragraph width
  4. rem - for closer to absolute value
  5. px - last resort for true absolute value

Margin

  1. rem - for closer to absolute value
  2. em - to scale with font size
  3. px - last resort small value

Borders and shadows

  1. px - for small values (preferred as they wont change much)
  2. rem/em - okay (But scaling wont look great)

Font size

  1. rem - usually the best
  2. em - Scale to parent size
  3. px - last resort
