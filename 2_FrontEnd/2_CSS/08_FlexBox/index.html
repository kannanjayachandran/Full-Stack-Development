we need to set the FlexBox property to the parent element.

FlexBox would only affect direct children not nested children

The default behavior of FlexBox is that the contents would take the size of their content

We can use flex-direction to specify the flex direction. By default we would consider the row as the main axis.

we can use justify-content to position the item along the axis.

We can use the align-items to position items along the cross axis.

We can use gap to give space between items.

We can use flex-wrap to stop overflowing from the FlexBox.

We can use the align-content property to determine how close each rows are with one another

We can use order to group items.

We can use flex-grow to specify how much the container can grow if there is space.
